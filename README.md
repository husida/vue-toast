# vue-toast

> A Vue.js plugin

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```

## toast-use
```
import toast from 'hu-vue-toast'
Vue.use(toast)

this.$toast('哈哈哈')
this.$toast('哈哈哈','top')// 'top','center','bottom'
```
